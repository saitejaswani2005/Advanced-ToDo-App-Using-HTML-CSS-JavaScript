# 📝 Advanced To-Do App Using HTML, CSS & JavaScript

This project is a basic yet functional To-Do web application that helps users manage their daily tasks efficiently. Users can add tasks, mark them as complete, edit or delete them, and view tasks in separate Pending and Completed sections.

## 🚀 Features

- ➕ Add new daily tasks
- 👀 View added tasks instantly
- ✅ Mark tasks as Completed
- 📂 Separate sections:
  - Pending Tasks
  - Completed Tasks
- ✏️ Edit tasks anytime
- 🗑️ Delete tasks from any list
- 🕒 Display date and time when:
  - Task is added
  - Task is completed
- 💾 Data stored using Local Storage (optional enhancement)

## 🛠️ Technologies Used

- HTML – Structure of the web app  
- CSS – Styling, layout, and responsiveness  
- JavaScript – App logic and interactivity  

## 🧠 Concepts Used

## 📌 How It Works

- User enters a task and clicks Add.
- Task appears in the Pending Tasks section with the current date & time.
- Clicking Complete moves it to the Completed Tasks section with completion time.
- Users can Edit or Delete tasks from either section.
- The To-Do list = Pending Tasks + Completed Tasks.
